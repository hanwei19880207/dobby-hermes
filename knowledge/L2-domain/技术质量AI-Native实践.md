## 各域实践亮点

### 正逆向交易（汪泽奇）
- 统一 Skill 方案：主 Skill + 分层知识上下文（最终为多 Agent）
- 知识上下文建设标准已明确
- 试点：结算页通下需求 AI Native 交付

### 大前端（丁鹏）
- 负责 Harness 工程标准和 Skill 建设规范
- 试点：营销频道超级明星 AI Native 交付（客户端）

### 本地生活服务（汪世涛）
- 用例采纳率 40-50%
- 文本用例第一版准确率 70%+
- 用例测新沉淀复用 → Harness 工程方式
- 推进架构组能力 CLI 化

### 安全风控（张栋珀）
- 风控规则引擎复核：30-60min → 3-5min
- Skill 保鲜机制：定期扫描引擎工程 + 复核反馈闭环
- RESTful API 开放设计

### 客服（程超）
- 完整测试流程：PRD质检 → TRD质检 → 代码变更分析 → 测试分析 → 用例生成 → UI自动化
- 质量流程嵌入研发流程

### 营销商品（周璇）
- 8 个 Skill 试跑调试中（PRD质检-代码分析-防呆扫描-测试分析-用例生成-物料生成-UI自动化）
- 防呆 Skill 已在研发阶段使用
